dna.js

An uncomplicated user interface library for building data-driven semantic templates (written in TypeScript)

Fork me on GitHub
Project:

https://dnajs.org

dna.js is an easy-to-use UI library enabling developers to rapidly build maintainable JavaScript applications.  You write semantic templates and then dna.js converts your JSON data into dynamically generated DOM elements.
